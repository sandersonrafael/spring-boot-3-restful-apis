@RestController -> declara que trata-se de um controller e o spring abstrai
@RequestMapping -> define o caminho para acessar o controller na url do navegador
@PathVariable -> parâmetros que são fornecidos obrigatoriamente na url como caminhos


# Exceptions

@ResponseStatus(HttpStatus.OPÇÃO) -> Declara que trata-se de um erro do spring e qual o código receberá
    -> usado na exceção personalizada

# Utilizados no exception handler
@ControllerAdvice -> declara que trata-se de um controller de aviso de erro
    -> o exception handler que receber essa anotação, se torna o controller de erro padrão para todos
        os demais controllers que não possuírem outro handler específico associado
@RestController -> declara que é um controller
    extends ResponseEntityExceptionHandler -> é o handler default dos controllers no spring
@ExceptionHandler(Exception.class) -> O método do handler de exceção em questão trata todas exceções genéricas
